# Penerapan-Association-Rule-Mining-untuk-Identifikasi-Pola-Kejahatan-pada-Data-Kriminalitas-Chicago-
Proyek ini menganalisis pola kriminalitas Kota Chicago tahun 2025 menggunakan teknik Association Rule Mining. Tujuannya adalah menemukan pola tersembunyi antara jenis kejahatan, lokasi, waktu, dan status penangkapan
